# 🌀 MeetFlow · 智能会议助手

> AI-powered meeting assistant focused on pre-meeting preparation and post-meeting follow-up.  
> 聚焦会前准备与会后跟进的轻量级 AI 会议助手。

## 🔗 在线体验

👉 [点击访问 Live Demo](https://meetflow-seven.vercel.app)

## 💡 产品背景

职场中开会最普遍的两个断层：
- **会前**：不知道准备什么，没有议程，目标不清晰
- **会后**：口头承诺没有记录，不知道谁做什么，纪要整理耗时

MeetFlow 专门解决这两个断层。

## ✨ 核心功能

**会前准备**
- 🗓 AI 智能议程生成：输入主题，一键生成结构化议程
- 🎯 会议目标设定：明确本次会议要达成什么
- ✅ AI 准备建议：个性化的会前准备清单
- 📋 会议模板库：周会 / 启动会 / 复盘 / 评审一键套用

**会后跟进**
- 📄 AI 结构化纪要：自动整理决议、讨论要点、遗留问题
- ✅ AI 待办提取：自动识别责任人与截止日期
- ⚠️ AI 风险提醒：主动发现无负责人、deadline 冲突等风险
- ⭐ 会议效率评分：量化每次会议的质量

## 🛠 技术实现

- 纯前端实现，无需后端，零依赖
- 单文件 HTML + CSS + JavaScript
- 模拟 AI 生成逻辑，展示完整产品结构与交互设计
